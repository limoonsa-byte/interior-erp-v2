### 견적서 – 특이사항(비고)
- **특이사항 입력**: 라벨 "특이사항 (비고)", textarea로 여러 줄 입력 가능, placeholder 안내 문구 추가
- **템플릿**: 템플릿 불러오기 시 `note` 반영, 템플릿으로 저장 시 `note` 포함 (POST/PATCH)
- **엑셀**: 내보내기 5행 라벨 "특이사항", 가져오기 시 5행 값으로 `note` 설정
- **미리보기·인쇄**: `EstimatePreviewSpecialNotes` 컴포넌트로 특이사항 표시. 입력이 있으면 줄별 bullet, 없으면 *특이사항 블록 미표시

### API
- **company/estimate-templates**: GET/POST/PATCH에 `note` 포함. GET 응답을 `rowToTemplate` 함수로 변환해 구문 오류 제거
- **company/default-estimate-templates**: 동일하게 `note` 포함, `rowToTemplate` 적용
- **DB**: `company_estimate_templates`, `master_default_estimate_templates` 테이블에 `note` 컬럼 추가 (`scripts/migrate.js`)

### 견적서 미리보기·인쇄 테이블
- **소계·공과잡비·이윤**: 메인 견적표(6열) 안에 같은 표로 행 추가. 표기 "소     계", "이     윤" (띄어쓰기)
- **정렬**: 순번·단위·수량·비고 헤더 및 해당 셀 중앙 정렬. 소계/공과잡비/이윤 품목 셀 중앙 정렬. 금액 우측 정렬
- **열 비율**: 순번 7%, 품목 30%, 단위 7%, 수량 7%, 금액 30%, 비고 19% (`colgroup` + `globals.css` estimate-preview-summary-table)
