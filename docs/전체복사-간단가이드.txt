========================================
전체 복사 간단 가이드
========================================

1. 기존 code.gs 파일 전체를 복사 (Ctrl+A, Ctrl+C)

2. 새 파일에 붙여넣기

3. doGet 함수 찾기 (맨 위)

4. doGet 함수 전체를 삭제하고, 
   "doGet-수정본.txt" 파일 내용으로 교체

5. 파일 맨 아래 (setupPriceTables 함수 뒤)에
   "getDrawingListForErp_-추가할함수.txt" 파일 내용 추가

6. 완료! 이제 전체를 복사해서 code.gs에 붙여넣기

========================================
또는 더 간단하게:
========================================

1. "doGet-수정본.txt" 파일 열기 → 전체 복사
2. 기존 code.gs에서 doGet 함수만 교체
3. "getDrawingListForErp_-추가할함수.txt" 파일 열기 → 전체 복사
4. 기존 code.gs 맨 아래에 추가
5. 완료!

이렇게 하면 기존 코드는 그대로 유지되고 필요한 부분만 수정됩니다.
